## Project Snapshot

`qt-llmlite` is a lightweight Qt/C++ LLM integration library.

Current baseline includes multi-client/session persistence, profile-driven context injection,
tool runtime orchestration, internal tool-loop handling in `QtLLMClient`, and vendor protocol mapping.
