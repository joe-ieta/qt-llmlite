# v0.2.2

This version focuses on documentation cleanup and project positioning clarification.

Highlights:

- rewrote the project introduction to clearly position `qt-llm` as a lightweight Qt-based LLM interface layer
- clarified the three-layer repository structure:
  - `qtllm` as the core infrastructure layer
  - `apps` as practical tool and host applications
  - `agents` as business-agent implementations and reference projects
- emphasized direct code-level integration into user Qt projects
- strengthened the description of runtime monitoring, tracing, and analysis support for conversation, Tools, MCP, SKILL, and Agent workflows
- simplified the active agent documentation set and archived older design-stage documents by version

This is a documentation and structure-alignment release. It does not introduce a new product feature baseline.
