# README / 项目说明

## 摘要 / Summary
qt-llmlite 是一个面向 Qt 开发者的轻量级本地模型接入库。项目支持以 Qt 包方式复用，也支持源码级集成。

## 介绍
qt-llmlite 基于 Qt/C++ 提供统一客户端与 Provider 抽象，用于快速连接 Ollama、vLLM 及其他 OpenAI 兼容接口。

## 核心价值
- 为 Qt 应用提供本地模型快速接入路径
- 降低现有工程改造成本
- 保持 Windows/Linux 开发一致性

## 集成方式
- 作为独立 Qt 包引入
- 作为源码模块直接集成
