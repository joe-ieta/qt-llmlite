# README / 砐醴佽隴

## Summary / 晡猁
qt-llmlite is a lightweight local-LLM integration library for Qt developers. It can be adopted as a Qt package or integrated at source level.

## Introduction
qt-llmlite provides a Qt/C++ native client and provider abstraction for quickly connecting local model services (such as Ollama and vLLM) and OpenAI-compatible APIs.

## Core Value
- Fast local-model integration for Qt apps
- Low refactoring cost for existing projects
- Consistent Windows/Linux development experience

## Integration Modes
- Standalone Qt package
- Source-level integration
