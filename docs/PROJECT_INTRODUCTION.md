# 项目简介 / Project Introduction

## 中文

`qt-llmlite` 是一个面向 Qt 开发者的轻量级本地模型接入库。

核心定位：
- 为 Qt 应用提供本地模型快速接入方法
- 保持 Qt/C++ 原生开发体验
- 通过 Provider 抽象兼容 OpenAI 风格接口

支持方式：
- 作为独立 Qt 包发布与复用
- 作为源码级模块直接集成到现有工程

典型价值：
- 降低本地大模型接入门槛
- 减少业务工程改造成本
- 保持跨平台（Windows/Linux）开发一致性

## English

`qt-llmlite` is a lightweight local-LLM integration library for Qt developers.

Core positioning:
- Provide a fast local-model integration path for Qt applications
- Preserve a native Qt/C++ development workflow
- Use a provider abstraction compatible with OpenAI-style APIs

Adoption modes:
- Publish and reuse as a standalone Qt package
- Integrate directly as source code in existing projects

Key value:
- Lower the integration barrier for local LLM capabilities
- Reduce refactoring cost for product teams
- Keep cross-platform consistency (Windows/Linux)
